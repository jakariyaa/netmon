# xfce4-genmon-goodies

Collection of small utilities scripts for Xfce Generic Monitor Plugin.

**Projects**
- **netmon**: Network monitor plugin. See [netmon README](netmon/README.md) and source at [`netmon/netmon.cpp`](netmon/netmon.cpp).
- **tempmon**: Temperature monitor utility. See [tempmon README](tempmon/README.md) and source at [`tempmon/tempmon.c`](tempmon/tempmon.c).

**Contributing**
- **New Plugins**: Contributions of new Genmon plugins are welcome!
- **Issues & PRs**: Please open issues or pull requests for bug fixes and improvements.
